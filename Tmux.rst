####
Tmux
####

:code:`^a  arrow`  : switch pane

:code:`^a + arrow` : resize pane

:code:`^a {` : move panes around

:code:`^a }` : move panes around

:code:`^a "` : split pane horizontally

:code:`^a %` : split pane vertically

:code:`^a ?` : help!

:code:`^a !` : break pane

:code:`^a @` : join pane
